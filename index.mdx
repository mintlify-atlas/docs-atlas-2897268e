## The AzuraCast community node for n8n

`@renebello/n8n-nodes-azuracast` is an n8n community node that connects [AzuraCast](https://www.azuracast.com/) — the self-hosted web radio platform — to your n8n workflows through the official AzuraCast Web API.

With a single node and 263 operations organized by resource, you can automate station management, media uploads, playlist changes, live broadcasting controls, reports, and full admin operations.

## Key features

<CardGroup cols={3}>
  <Card title="263 / 263 operations" icon="circle-check">
    Full coverage of every operation in the official AzuraCast OpenAPI spec.
  </Card>
  <Card title="Single unified node" icon="cube">
    One node in your canvas — pick a resource tag, then pick an action.
  </Card>
  <Card title="All request types" icon="arrow-right-arrow-left">
    JSON body, raw body, binary upload, and multipart form-data are all supported.
  </Card>
  <Card title="Smart response handling" icon="bolt">
    Auto-detects JSON, binary, or text responses per endpoint with manual override.
  </Card>
  <Card title="Public & protected endpoints" icon="lock-open">
    Public endpoints work without a key. Protected endpoints add auth headers automatically.
  </Card>
  <Card title="Backward compatible" icon="clock-rotate-left">
    Legacy `operationId` and JSON parameter workflows from earlier versions continue to work.
  </Card>
</CardGroup>
