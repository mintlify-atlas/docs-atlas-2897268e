The AzuraCast node connects n8n workflows to the full AzuraCast Web API. All 263 operations across 27 resource categories are available from a single node.

## How it works

<Steps>
  <Step title="Add the node to your canvas">
    Search for **AzuraCast** in the n8n node panel and drag it onto your workflow canvas.
  </Step>
  <Step title="Configure credentials">
    Select or create an **AzuraCast API** credential. You need a Base URL (e.g. `https://radio.example.com`) and, for protected endpoints, an API key.
  </Step>
  <Step title="Select a Resource">
    Choose the API domain you want to work with, such as **Stations Media**, **Stations Playlists**, or **Administration Users**. Resources correspond to official AzuraCast API tags.
  </Step>
  <Step title="Select an Operation">
    Pick the specific endpoint (action) within that resource. The node subtitle in n8n updates to show the current operation name, making it easy to identify at a glance what each node does in a larger workflow.
  </Step>
  <Step title="Fill in the fields">
    Only the fields relevant to the chosen operation are shown — path parameters, required query/body fields, and optional collections. Fill in the required inputs and expand optional sections as needed.
  </Step>
  <Step title="Execute">
    Run the node. The response is automatically parsed and returned as JSON, binary, or text output depending on the endpoint.
  </Step>
</Steps>
